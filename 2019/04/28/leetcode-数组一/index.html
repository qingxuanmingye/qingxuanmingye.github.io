41. 缺失的第一个正数（B+）

分析：

注意：你的算法的时间复杂度应为O(n)，并且只能使用常数级别的空间。

（1）首先把数组内，范围在 ( 0, n ] 元素x，放到nums[x - 1]，类似桶排序中的找位置。
（2）放好之后，进行遍历，找第一个 i + 1 != nums[i]，返回 i + 1，就是结果，如果没有就返回 n + 1。

（3）所以，最小的正整数很重要，如果是第一个没出现的数，就不能这样做了。
